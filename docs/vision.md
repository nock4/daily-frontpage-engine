# Vision

The vision is a new dynamic front page every single day.

This is not one homepage.
This is a daily scene engine that emits a new AI-generated interactive artwork every day.

Core rules:
- every day gets a new scene
- never reuse the same scene twice
- keep an explorable archive of past editions
- optimize for mood and world first
- prefer abstract, image-led, research-shaped worlds over default office/desk metaphors
- the page should feel like a living interactive artwork
- research should deepen the signal field before generation
- interesting source images should be used as generation inputs when they materially improve composition, texture, or formal language
- source windows should expose real media and real sources, not text-summary cards
- public source bindings should come from recent saved-source signals, not unrelated personal vault notes or arbitrary web discoveries
- NTS likes should resolve to streamable sources before becoming pockets
- YouTube pockets should be playable embeds in generated editions
- tweet media should support the visual presentation, while native tweet URLs remain the source of truth

The image is not decoration.
The image is the interface.
