# Daily Frontpage Engine - Still-in-Play Roadmap

Status: historical roadmap from April 24. Use `docs/current-state.md`, `docs/process.md`, and `docs/repo-roadmap.md` as the current source of truth.

## What Landed Since This Roadmap

- Edition-scoped `about.json` records are generated for new packages.
- The daily pipeline is runnable with one command: `npm run daily:process`.
- From-scratch runs now include saved-signal mining, source autoresearch, browser-harness capture, `gpt-image-2` plate generation, `gpt-5.5` post-plate inspection, package assembly, source image enrichment, mask generation, interpretation/enhancement generation, validation, unit tests, build, Playwright smoke, and source-window media audit.
- Source selection is now policy-driven: 7-10 renderable non-duplicate sources, NTS resolved streaming sources, YouTube embeddability checks, and no raw Twitter/X CDN primary URLs.
- The current visual direction is more minimal and abstract-expressionist.

## Current Open Work

- Decide whether the latest generated review edition should be promoted live.
- Improve the scene-native presentation of About content without adding product chrome.
- Continue tightening contour masks for abstract plates that produce broad field-like masks.
- Improve Bandcamp/SoundCloud/audio source-window treatments.
- Deepen ambiance/reactivity only where it reinforces the plate and source truth.

## Historical Intent That Still Applies

- The image is the interface.
- Live mode should remain artwork-first.
- Source windows should expose real media, not summaries.
- Daily generation should be repeatable and reviewable.
- Pipeline automation is valuable only when it preserves editorial review and source truth.
